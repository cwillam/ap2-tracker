02Wie kann man die App einsetzen

Ein paar bewährte Szenarien für den Unterrichtsalltag:

Flipped Classroom

Schüler lernen die Prüfungsthemen zu Hause mit den Lernkarten, im Unterricht wird vertieft und diskutiert. Funktioniert, weil man ohne Anmeldung sofort loslegen kann.

Präsenzunterricht am Beamer

Lehrer zeigt eine Frage über den Beamer oder das Whiteboard, die Klasse diskutiert und löst gemeinsam. Funktioniert auf jedem Gerät mit Bildschirm.

Selbststudium zu Hause

Schüler wiederholen eigenständig, die App speichert alles im Browser. Kein Account, keine Mail, keine Bestätigungsmails an die Eltern.

Vertretungsstunden

Funktioniert ohne Lehrer-Anleitung. Schüler können selbstständig üben, weil die Bedienung selbsterklärend ist.

03Wo die Daten bleiben

Das ist mir wichtig. Hier der genaue Datenfluss, damit du das deinen Schülern erklären kannst, falls die fragen:

Browser LocalStorage // Lernfortschritt: abgehakte Karten, Streak, Timer
↓ HTML + JS // läuft komplett im Browser, kein Server-Code
↓ IONOS (EU) // liefert nur die HTML-Dateien aus, Server-Logs 7 Tage
↓ an Dritte // geht nirgendwo hin

Keine Benutzer-Accounts, keine Cookies, keine externen CDNs, keine Google Fonts, keine Analyse-Tools. Wer das prüfen will: Datenschutzerklärung oder einfach in den Browser-Devtools unter „Network" schauen, während die Seite offen ist.
